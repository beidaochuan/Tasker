# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Removed
- タスクのキャンセルステータスとカンバンのキャンセル列を削除

## [0.8.0] - 2026-07-14

### Changed
- カンバンの未着手・進行中タスクを優先度順、完了・キャンセル済みタスクを状態変更日時の新しい順に表示するよう変更
- リストのタスクを優先度順に統一し、手動並び替えを同じ優先度内に制限
- ライト・ダークテーマの面、境界線、優先度、エラー表示のコントラストを改善

### Fixed
- カンバンカードのキーボードフォーカスを明示し、ドラッグ終了・キャンセル後に枠やオーバーレイが残る問題を修正

## [0.7.0] - 2026-07-14

### Changed
- 起動時の既定ビューをカンバンに変更し、ビュータブを「カンバン、ガント、リスト、カレンダー」の順に整理
- 初回利用時の既定テーマをダークに変更
- カンバンカードのタイトルと優先度を強調し、説明文を省いて情報密度を調整
- ガントのタスク一覧を拡張し、優先度ラベルをタイトル前の上下中央に表示するよう改善
- リストとガントバーの重複する優先度表示を整理し、ガントの完了タスクグループの通常時の主張を抑制

### Fixed
- タスクとプロジェクトの作成・編集画面を、IME変換や作業項目の編集中を考慮しながらESCで保存せず閉じられるよう修正

## [0.6.0] - 2026-07-13

### Changed
- リストとガントの完了タスクを、トピックごとの件数付き折りたたみグループにまとめるよう変更
- 完了タスクグループの開閉状態をリストとガントで共有するよう変更
- タスクの並び替えを未完了・完了それぞれのグループ内に制限し、非表示タスクを含む全タスクの順序を維持するよう改善

## [0.5.1] - 2026-07-13

### Fixed
- 配布ZIPで `npm ci --omit=dev` を実行すると、同梱されていないHusky初期化スクリプトの読み込みに失敗する問題を修正

## [0.5.0] - 2026-07-13

### Added
- ガントビューに完了タスクの件数付き表示切替を追加

### Changed
- ガントビューでは完了タスクを初期状態で非表示にするよう変更
- リストビューでは完了タスクを各トピックの末尾に表示するよう変更
- ガントで完了タスクを非表示にしたまま並び替えても、全タスクの表示順序を維持するよう改善

## [0.4.0] - 2026-07-11

### Added
- タスク詳細に、項目の追加・名称編集・完了切替・削除ができるチェックボックス付き作業リストを追加
- タスク編集時にプロジェクトとトピックを変更できる機能を追加
- ガントビューに曜日表示とタスクのドラッグ並び替えを追加

### Changed
- 起動時にプロジェクト一覧の先頭を自動選択するよう変更
- GitHub Releases からWindowsサービスとして利用するまでの手順をREADMEに追加

### Fixed
- タスク編集時、トピック読み込み中に既存の選択が上書きされる問題を修正
- リストとガントでタスクを並び替えた際の表示のちらつきや順序不整合を修正
- ガントの並び順更新をトランザクションで一括保存するよう修正
- 作業リストの日本語入力確定時に項目が追加・更新される問題を修正

## [0.3.4] - 2026-07-06

### Fixed
- IP アドレスでアクセスした場合に CORS エラーで API リクエストが拒否される問題を修正（`CORS_ORIGIN` 未設定時は全オリジン許可に変更）

## [0.3.3] - 2026-07-06

### Fixed
- サーバーが全ネットワークインターフェース（0.0.0.0）でリッスンするよう修正し、同一ネットワーク内の別端末からアクセスできない問題を解消

## [0.3.2] - 2026-07-06

### Added
- Windows サービスへの登録・削除スクリプトを追加 (`npm run service:install` / `npm run service:uninstall`)
- `node-windows` パッケージを依存関係に追加

## [0.3.1] - 2026-07-04

### Fixed
- GitHub Release の配布物に API サーバーを同梱し、静的ファイル単体では動作しない問題を修正
- JSON インポートのサーバー側リクエストサイズ上限を 50 MB に合わせるよう修正
- 繰り返しタスク完了時の完了履歴記録と次回タスク作成をトランザクション化
- タグ削除時にタスク内へ残る孤立タグ ID を削除するよう修正
- API の 400 / 409 系エラーを validation / conflict として扱うよう修正

### Changed
- ビルド時に API サーバーも型チェックとビルドの対象に含めるよう変更

## [0.3.0] - 2026-06-16

### Added
- 簡易ログインによる編集モード切替
- 未ログイン時の閲覧モード
- 未ログイン時の作成・編集・削除・ドラッグ更新・インポート操作の制限

## [0.1.0] - 2026-06-12

### Added
- プロジェクト・トピック・タスクの作成・編集・削除
- タスクのリストビュー（DnD によるソート・フィルタ対応）
- カンバンビュー（列間ドラッグ＆ドロップ）
- カレンダービュー（FullCalendar ベース）
- ガントビュー（バーのドラッグ・リサイズ対応）
- 繰り返しタスク（RRule ベース、全発生日展開表示）
- タグ管理機能
- JSONエクスポート・インポート
- PWA 対応（オフライン利用可能）
- ダークモード対応
- プロジェクト削除（トピック・タスクの連鎖削除）
- タスク一覧の削除ボタン（ホバー表示・確認ダイアログ）
